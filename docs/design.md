# miraclepopcorn デザイン方針

## サイトの性格

miraclepopcornは、日常、道具、仕組み、写真、ものづくりについて記録する、静かな個人サイトです。

## 優先事項

1. 初見で理解しやすい
2. 読みやすい
3. 写真が映える
4. 静かで整然としている
5. 将来の拡張に対応できる

## ビジュアル方針

- 余白を広めに取る。
- タイポグラフィを重視する。
- 明るい背景と暗い文字を基本にする。
- 線や境界表現は控えめにする。
- 影は弱くするか、原則として使わない。
- 角丸は弱めから中程度にする。
- 少し工業的で構造的な印象を持たせる。
- レトロフューチャー感は控えめにする。
- 写真そのものを主役にする。

既存UIでは、温かみのある明るい背景、暗い本文色、控えめな境界線、広めの余白、整理された本文幅を基調としている。新しいUIを追加するときも、この表現を優先して既存のコンポーネントやスタイルと矛盾しないようにする。

## 避けるもの

- スタートアップのランディングページのような表現
- 情報密度の高いテックメディア
- 柔らかすぎる生活ブログ
- 過剰な装飾
- 強すぎる懐古趣味
- ポートフォリオ的な強い自己主張

## ページごとの性格

- **Home**: 個人ポータルの入口
- **Blog**: 静かで整理された記事一覧
- **Article**: 長文と写真の両方に対応する読み物
- **Profile**: 簡潔で文章中心のページ
